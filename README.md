## Descripción

Este ejercicio consiste en representar mediante un diagrama de flujo el programa desarrollado en el Deber 1.

El programa realiza una conversión de divisas ingresando:

- divisa base
- divisa cotizada
- tasa de cambio
- monto base
- timestamp

Luego calcula el monto convertido y muestra los resultados.

## Lógica del programa

1. El usuario ingresa la divisa base.
2. Se valida que tenga 3 letras y que sean caracteres alfabéticos.
3. El usuario ingresa la divisa cotizada.
4. Se valida que tenga 3 letras válidas.
5. El usuario ingresa la tasa de cambio.
6. Se valida que sea un número mayor a cero.
7. El usuario ingresa el monto base.
8. Se valida que sea un valor positivo.
9. El usuario ingresa un timestamp.
10. El programa calcula el monto convertido multiplicando el monto base por la tasa.
11. El programa muestra los resultados de la conversión.
12. El usuario puede elegir repetir la operación o terminar el programa.
